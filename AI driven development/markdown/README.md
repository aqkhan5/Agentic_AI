## 🛠️ Core Capabilities

### 1. Custom Blog Planner Skill (`blog-planner`)
The workspace includes a custom Claude Code skill, configured in `.claude/skill/blog-post/skills.md`, to systematically outline and structure blog posts.

#### **Procedure**
1. **Understand Topic & Target Audience**: Define who the post is for and what problem it resolves.
2. **Create Outline**: Structure content using a problem-solution narrative model (Problem → Failed Solutions → Insight → Application).
3. **Generate Headlines**: Design 5 curiosity-driven, benefit-focused, and direct title variations (no clickbait).
4. **Draft Introduction**: Write a compelling introduction hook that challenges assumptions or poses unexpected questions.

#### **Standard Output Format**
- **Topic Summary**: 2-3 sentence overview.
- **Target Audience**: Profile of the ideal reader.
- **Outline**: A numbered list of main sections with brief descriptions.
- **Headline Options**: 5 distinct headlines.
- **Introduction Draft**: 1-2 paragraph hook.

## 📜 Communication & Academic Guidelines (`CLAUDE.md`)

All content generated in this workspace must adhere to the professional standards defined in `CLAUDE.md`:
- **Formal Academic Tone**: No colloquialisms, contractions, or informal expressions.
- **Objective Perspective**: Neutral third-person exposition supported by logical reasoning.
- **Precise Terminology**: Accurate use of technical terms, defined on first introduction.
- **Evidence-Based Reasoning**: Assertions must be grounded in facts, examples, or logical arguments.
- **Analytical Depth**: Detailed analysis of trade-offs, implications, and alternatives.
